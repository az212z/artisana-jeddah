## 2) مخرجات design-system المعتمدة (palette / typography)

- **Palette (semantic tokens):**
  - `--navy #0E2236` (الأساس) · `--cream #F6F1E7` (الخلفية) · `--brass #C9A24B` (الأكسنت)
  - نص: `--on-bg #21303C` على الكريمي (~9:1)، `--on-navy #F3EEE2` على الكحلي.
  - الأكسنت **مقفول** (نحاسي واحد عبر كل المقاطع)، لا hex مبعثر.
- **Typography:** عناوين **El Messiri** 600/700، نصوص **Tajawal** 400/500/700. مقياس 12→48، line-height 1.7 للجسم.
- **Effects:** هيرو كحلي بتدرّج نحاسي radial خفيف، حواف 8/14/22، أزرار pill، lightbox، FABs.

> **مبرر استخدام display "serif-like":** El Messiri مفروض بالبريف صراحةً، والطابع فاخر/تراثي حقيقي — فالاستثناء في قاعدة "تجنّب السيرف الافتراضي" منطبق ومُبرَّر.

## 3) قرارات UI/UX الأساسية
- **مسار التحويل مُعاد تصميمه للواقع:** لا واتساب (الرقم ثابت). الـ CTA الأساسي = **نموذج عرض سعر/استشارة**، والثانوي = **اتصال مباشر**. الأزرار العائمة = اتصال + خرائط فقط.
- بنية AIDA: هيرو ← شريط ثقة ← خدمات ← لماذا نحن ← معرض ← نموذج ← الموقع ← CTA ختامي ← فوتر.
- تنويع تخطيطي: services grid (3-col)، why (feature rows ترقيمية)، gallery (bento: عنصر طويل + عنصران)، quote (split نموذج)، location (3 info-cards) — عائلات مختلفة، بلا تكرار.

## 4) سبب الألوان والخطوط
- **الكحلي + النحاسي + الكريمي:** يعبّر عن الفخامة الكلاسيكية المطلوبة، ويميّز الموقع عن الشقيق "البسيط الفحمي". النحاسي يربط بصريًا مع الذهبي في شعار أرتيزانا الحقيقي.
- **El Messiri + Tajawal:** ثنائي عربي (عناوين أنيقة + جسم نظيف) يليق بنشاط تصميم داخلي راقٍ.

## 5) تطبيق Hooked / UX سلوكي
- CTA واحد مهيمن في كل مشهد، تكرار نقطة التحويل (هيرو/فوتر/CTA ختامي/FAB) بلكنة واحدة.
- مكافأة فورية: toast نجاح + حالة زر "تم الإرسال ✓" بعد الإرسال.
- احتكاك منخفض: نموذج قصير (اسم/جوال/خدمة/ملاحظات) فقط.

## 6) تطبيق iOS HIG / تفاعل
- أهداف لمس ≥48px (أزرار، حقول، FABs، burger). تباعد مريح.
- ردّ فعل ضغط <150ms (`scale(0.97)`) بلا إزاحة layout.
- النموذج: تحقّق عند الإرسال، خطأ تحت الحقل مباشرة، focus على أول حقل غير صالح.

## 7) تطبيق Accessibility (تم التحقق)
- `<html lang="ar" dir="rtl">`، HTML سيمانتيك (header/nav/main/section/footer)، تدرّج h1→h3 بلا قفز.
- تباين: نص على كريمي ~9:1، نص على كحلي ≥7:1، الأكسنت النحاسي على الكحلي للعناصر الكبيرة. **كل الأزواج ≥4.5:1.**
- `:focus-visible` بـ outline نحاسي 3px على كل عنصر تفاعلي. `skip-link`. كل أيقونة-زر لها `aria-label`. كل صورة لها alt عربي وصفي + width/height + lazy لغير الهيرو.
- القائمة dialog/aria-modal، Escape يغلق، إدارة focus. `prefers-reduced-motion` يوقف الحركة والتمرير الناعم.
- اللون ليس الناقل الوحيد (أيقونات + نص مع كل دلالة).

## 8) Impeccable / Taste (اختبار القبول)
- هل يبدو فاخرًا؟ نعم — كحلي/نحاسي، El Messiri، صور أعمال حقيقية، ظلال ومسافات راقية.
- سعودي مناسب للنشاط؟ نعم — نصوص طبيعية محايدة، تقييم قوقل، رقم ثابت محلي.
- يقنع خلال 3 ثوانٍ؟ نعم — عنوان واضح + CTA + صورة + تقييم فوق الطية.
- لا يشبه قالبًا مجانيًا؟ نعم — هوية مخصّصة، bento معرض، أرقام feature، favicon من حرف الشعار.

## 9) التحقق الفعلي
- مُعايَن على متصفح حقيقي (preview): الهيرو، القائمة ملء الشاشة (خلفية كحلية صلبة + X)، قسم الخدمات.
- **Playwright: 20/20 ناجح** (ديسكتوب + جوال) — منها: لا تمرير أفقي @390px، لا واتساب، القائمة ملء الشاشة، تحميل كل الصور، JSON-LD.
