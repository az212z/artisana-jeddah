# أرتيزانا إنترناشونال · موقع تعريفي

موقع تسويقي فاخر (عربي RTL) لشركة **أرتيزانا إنترناشونال** للتصميم الداخلي والديكور في جدة.

## معلومات النشاط
- **الاسم:** أرتيزانا إنترناشونال (ARTISANA International)
- **التصنيف:** شركة تصميم داخلي وديكور وأثاث فاخر
- **المدينة:** جدة · حي الزهراء
- **التقييم:** 4.4 على خرائط قوقل (24 تقييمًا) — مذكور في الموقع وفي بيانات JSON-LD
- **الهاتف:** 0122753477 (هاتف ثابت — landline)

## لماذا اختير هذا النشاط
نشاط تصميم داخلي بتقييم جيد وصور أعمال حقيقية، وبدون موقع إلكتروني — فرصة مناسبة لموقع تعريفي راقٍ يبرز الأعمال ويولّد طلبات استشارة.

## فكرة التصميم
هوية **"كلاسيكية فاخرة"** تميّزها عن الأشقاء البسيطة:
- ألوان: **كحلي عميق + نحاسي/ذهبي + كريمي دافئ**.
- خطوط: عناوين **El Messiri** + نصوص **Tajawal**.
- بنية تعتمد على عرض الأعمال (portfolio-led) مع تدرّج إقناعي AIDA.
- نصوص عربية طبيعية **محايدة جندريًا** بالكامل (اطلب/تواصل/شاهد — بلا صيغ مذكّر أو مؤنّث).

## مسار التحويل (مهم)
> النشاط يملك **هاتفًا ثابتًا فقط، بلا واتساب أو جوال**.
لذلك مسار التحويل الأساسي هو:
1. **نموذج "اطلب استشارة / عرض سعر"** (CTA رئيسي).
2. **زر اتصال مباشر** على الرقم الثابت 0122753477.

**لا يوجد زر واتساب إطلاقًا** (لأنه سيكون رقمًا وهميًا). الأزرار العائمة = **اتصال + خرائط** فقط.

## التقنية
- موقع **ثابت (vanilla)** بلا أي مرحلة بناء: HTML + CSS + JavaScript فقط.
- لا `node_modules` ولا `package.json` في المستودع.
- خطوط Google عبر preconnect و`display=swap`؛ أيقونات SVG inline.
- متجاوب موبايل-أولًا، قائمة جوال ملء الشاشة، scroll-reveal مع fallback، lightbox للمعرض.

## حالة الـ Backend
**غير متصل بخادم.** النموذج تجريبي للعرض:
- يتحقق من المدخلات، ثم يحفظ الطلب في `localStorage` بمفتاح `artisana_quotes`، ويُظهر **toast** نجاح.
- لا تُرسل أي بيانات لأي خادم. للتفعيل الحقيقي يُربط لاحقًا بنموذج/بريد.

## مصدر الصور
صور أعمال الشركة من ملفها على خرائط قوقل، تمت **مراجعتها وتنقيتها**:
- مُستبعد: صورة فيها وجوه واضحة لأشخاص، وصورة عليها شعار/علامة شركة منافسة (RM).
- المُعتمد: `art-13433` (صالة طعام)، `art-29370` (مجلس)، `art-6159` (تنفيذ/تشطيب)، إضافة إلى شعار أرتيزانا `art-31592`.
